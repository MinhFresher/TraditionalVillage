# Ước Lễ UI/UX Project

## Giới thiệu
Dự án này là một ứng dụng web được xây dựng bằng React, nhằm giới thiệu và quảng bá làng nghề truyền thống Ước Lễ. Ứng dụng cung cấp thông tin về lịch sử, sản phẩm, hoạt động và các giá trị văn hóa của làng.

## Cấu trúc dự án
- **src/pages**: Chứa các trang chính của ứng dụng:
  - `Homepage.jsx`: Trang chủ.
  - `IntroducePage.jsx`: Trang giới thiệu.
  - `Producpage.jsx`: Trang sản phẩm.
  - `Gallerypage.jsx`: Trang hình ảnh.
  - `Articlepage.jsx`: Trang bài viết.
  - `Activititiespage.jsx`: Trang hoạt động.
- **src/styles**: Chứa các tệp CSS để định dạng giao diện.
- **public/**: Chứa các tài nguyên tĩnh như hình ảnh.

## Công nghệ sử dụng
- **React**: Thư viện JavaScript để xây dựng giao diện người dùng.
- **React Router**: Quản lý điều hướng giữa các trang.
- **CSS**: Định dạng giao diện.

## Cách chạy dự án
1. Cài đặt các gói phụ thuộc:
   ```bash
   npm install
   ```
2. Chạy ứng dụng:
   ```bash
   npm run dev
   ```
3. Mở trình duyệt và truy cập: `http://localhost:3000`

## Tính năng chính
- Hiển thị thông tin về làng Ước Lễ.
